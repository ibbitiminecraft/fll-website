Core Values We've Learned

Throughout our journey in the First Lego League, we have learned some valuable lessons. Here are the key Core Values we've embraced:

  • Innovation: We use creativity and persistence to solve problems.
  • Impact: We apply what we learn to improve our world.
  • Gracious Professionalism: Competing with kindness and respect for others is essential.
  • Teamwork: We are stronger when we work together.
  • Discovery: We embrace challenges as opportunities to learn and grow.
